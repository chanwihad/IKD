# Manajemen-Indikator

## Requirement
- PHP Ver. 8
- Laravel Ver. 8
- NPM Ver. 8
- Composer Ver. 2

## Installation
- composer update
- php artisan migrate
- php artisan db:seed --class=OpdSeeder
- php artisan db:seed --class=UserSeeder
- php artisan config:cache
- Replace file didalam folder app/Tambahan direktori vendor/laravel/ui/auth-backend
